Amazon Web Scraper

I developed an Amazon.com scraper application that returns the top search results through a Discord bot, using Python and web scraping techniques with the BeautifulSoup package. The scraper is able to extract relevant product information from Amazon.com's front-end code such as the product title, price, image, ratings, number of ratings, and link to the product. The parsing algorithm was carefully designed to extract only the necessary information, making the output simple and easy to understand. The goal of the application is to provide a quick and efficient way for users to search for products on Amazon.com without having to navigate through the website themselves. This application is expected to save time and make the shopping experience more convenient for users who use Discord as a communication platform.
